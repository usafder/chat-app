# real-time-chat-app
A real-time chat app built using Vue, Firebase and Materialize CSS where users can chat with each other.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
